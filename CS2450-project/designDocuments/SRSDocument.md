Functional
1. UI displays the accumulator value of the VM
2. UI displays the program counter value of the VM
3. UI displays memory values of the VM
4. Program can execute one instruction at a time
5. Program can store a set of instructions in a file
6. Program can execute a set of instructions that has been loaded or entered by the user
7. VM can execute arithmetic instructions
8. VM can execute branching operations
9. The program handles all errors
10. The UI has navigation between the editor and the memory view
11. The program alerts the user if instructions do not conform to the UVSim standard
12. The terminal stops and waits when expecting user input
13. The terminal has a clear command to empty the output of previous operations
14. Terminal can display VM output to the user
15. The terminal passes values entered to the VM

Non functional
1. System must display the appropriate error messages when the user enters invalid input 100% of the time
2. When user clickes execute next or execute all, the program must execute the instructions in under 1 second 
3. User actions must be able to be completed in 3 clicks or less.
